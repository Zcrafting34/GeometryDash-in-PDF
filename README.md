
# Geometry Dash PDF Edition

Un clon experimental de *Geometry Dash* implementado completamente mediante **AcroForms** (formularios PDF).

##  Descripción
Este proyecto explora la capacidad de interactividad de los archivos PDF, convirtiendo un documento estático en un entorno de juego funcional.

###  Notas de compatibilidad
* **Visores recomendados:** El juego funciona correctamente en la mayoría de los visores de PDF integrados en navegadores modernos (como **PDFium** en Chrome, Edge o Firefox).
* **Adobe Acrobat:** Puede presentar errores de ejecución o limitaciones de script si no se utiliza la versión de pago (Acrobat Pro).

---

## 🚀 Guía de inicio

### 1. Jugar el juego
Para generar el archivo ejecutable del juego:
1. Asegúrate de tener instalado Python 3.x.
2. Ejecuta el archivo principal en tu terminal:
   ```bash
   python main.py


3. Se generará un archivo PDF en la carpeta `/build`. Ábrelo en tu navegador favorito para empezar a jugar.

### 2. Crear niveles personalizados

Si quieres diseñar tus propios retos:

1. Ejecuta el editor de niveles:
```bash
python level-editor.py

```

2. Diseña tu nivel siguiendo las instrucciones de la interfaz.
3. Al terminar, guarda el archivo generado dentro de la carpeta `/levels`.
